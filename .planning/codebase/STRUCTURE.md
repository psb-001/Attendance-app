# STRUCTURE

- `/app/`: Expo Router endpoints containing all screen views (e.g. `index.js`, `profile.js`, `student-dashboard.js`).
- `/components/`: Reusable UI modules (`SubjectCard.js`, etc.).
- `/context/`: Application-wide states (`ThemeContext.js`).
- `/lib/`: Service modules like `supabase.js` for initialization routines.
- `/data/`: Legacy JSON payloads for previous offline-first structure.
