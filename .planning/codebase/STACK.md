# STACK

## Core
- **Framework:** React Native
- **Framework Extension:** Expo (SDK ~54)
- **Routing:** expo-router (File-based navigation)
- **UI Library:** React Native Paper (MD3 styling)

## Backend
- **Database / Auth:** Supabase (`@supabase/supabase-js`)
- **Storage:** Local AsyncStorage (`@react-native-async-storage/async-storage`)

## Utilities
- **Icons:** @expo/vector-icons
- **Media:** expo-media-library, react-native-view-shot (Export capabilities)
- **Web support:** react-native-web (Often bundled for cross-compatibility)
